How to Join

  1. Open Minecraft Java Edition
  2. Click Multiplayer
  3. Click Add Server
  4. Enter Server Address: ApocalypseMC.com
  5. Click Join & survive.

Server Rules

  • No cheating or hacked clients.
  • No harassment or hate speech.
  • Raiding is allowed — but keep it fun.
  • Survive. Adapt. Don't give up.
